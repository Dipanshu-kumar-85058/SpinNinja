**Spin It** is a visually dynamic, browser-based random picker and matchmaker. Designed with a sleek dark-mode glassmorphic UI, it features cinematic 3D reveal animations, horizontal slot-machine-style spinning tracks, and a built-in confetti engine.

Whether you're picking a raffle winner, creating random project teams, or playing an elimination game, Spin It handles it with style.

## ✨ Features

* **🎯 Pick One (Single Mode):** The classic spinning wheel experience. Selects a single winner from your pool.
* **🎲 Pick Many (Multi Mode):** Need a top 3? Spin for up to 3 winners simultaneously with staggered track animations.
* **🤝 Pair Up (Matchmaker):** Create two separate lists (e.g., "People" and "Projects") and randomly pair them. Choose between picking a single random pair or matching everyone at once.
* **❌ Eliminate:** Survivor style. The chosen option is dramatically revealed as "Eliminated" and automatically removed from the active pool.
* **🎞️ Cinematic Reveals:** 3D pop-out result cards with staggered animations and custom-colored canvas confetti.
* **📜 Session History:** Automatically tracks and logs every roll with timestamps and the mode used.
* **📱 Fully Responsive:** Optimized grid layouts that scale perfectly from ultra-wide desktops down to mobile devices.
